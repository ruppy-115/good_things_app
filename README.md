## サービス概要
このアプリは、今日あった「良いこと」を記録できるアプリです。
記録の際に1日を振り返ることで今日をポジティブに捉えられるだけでなく、過去の投稿を見返すことで、何気ない日常が確かに積み重なっていることを実感できます。
さらに、世界中のユーザーの「良いこと」に触れることで、これまで認識できていなかった自分の「良いこと」に気づくことができます。

## このサービスへの思い・作りたい理由
このサービスの原点となったのは、東日本大震災を経験した語り部の方々による「生」の語りです。
それぞれの語りに共通していたのは「日々を大切に生きてほしい」という強いメッセージでした。

その言葉は、私の中に一つの疑問をもたらしました。
ーー「私は、1日1日を大切にできているだろうか。」

ある語り部の方は、「今日あった良いこと」を日記に書いていると話してくれました。
「今日あった良いこと」を記録することは、その日にあった出来事を踏み締め、日常の中にある無機質ではない何かを発見する行為であり、「日々を大切にすること」の一つの実践だと感じました。
一方で、日記という形は多くの人にとって、継続のハードルが高いのも現実です。
実際、私自身も日記を何度か始めたことがありましたが、習慣にすることは叶いませんでした。

そこで、毎日目にするスマートフォンのアプリという形態であれば、手帳等に書く場合と比べて、継続のハードルが低いのではないかと考え、このサービスを着想するに至りました。

## ユーザー層について
### 1日1日が足早に過ぎていくことに焦燥感を抱いている人
日々の生活が忙しく、1日を振り返る余白を持てないまま時間が過ぎていく感覚を抱く人は少なくないでしょう。
本サービスは、1日の終わりに「良いこと」を1つ記録するというシンプルな行為を通して、流れていく時間を「積み重なり」として捉え直すきっかけを提供します。
そのため、日常の速さに焦燥感を抱いている人ほど、本サービスの価値を実感しやすいと考えました。
### 気分が沈むことが多いが、前向きになりたいという思いを持っている人
気分が落ち込んでいるときほど、自分の生活の中にある肯定的な出来事に気づきにくくなります。
本サービスでは、無理に前向きになることを求めるのではなく、「良いこと」に意識を向けられるような設計を行なっています。
そのため、前向きになりたいという気持ちを持ちながらも、具体的な行動に移せていない人にとって、最初の一歩を踏み出しやすいサービスであると考えました。

## サービスの利用イメージ
①1日の終わりに、今日あった良いことを1つ入力
②過去の自分の記録を一覧で見返す
③世界の誰かの「良いこと」を読む
※③を起点に、他のユーザーの記録を見る→自分の一日を振り返る→今日あった「良いこと」を記録するという利用も想定している
└自発的な記録だけでなく、他者の記録をきっかけとした記録の導線を設けることで、無理のない継続を促す。

日常の中にあるポジティブな出来事への気づきと毎日の積み重なりを実感することができます。

## サービスの差別化ポイント・推しポイント
本サービスの推しポイントは、肯定的な出来事に気づくことで前向きな感覚を得られる点と、日常が確かに積み重なっていることを実感できる点の二つにあります。

一般的な日記アプリは、今日あった「こと」をそのまま書き残す形式が多く、内容が事実の羅列にとどまりがちです。本サービスでは、あえて記録対象を、今日あった「良いこと」に限定しています。これにより、ユーザーは自然と一日の中の肯定的な出来事に意識を向けることになります。自己表現を目的とするのではなく、日常に埋もれがちな肯定的な一側面に気づくことを大切にした設計です。

また、本サービスは自分の記録を残すだけで完結しません。世界中のユーザーの「良いこと」を閲覧できる仕組みを取り入れることで、他者の視点を通して、自分では気づけなかった日常の「良いこと」に気づくきっかけを提供します。一般的な日記アプリは、自分の記録のみを閲覧する形式が多く、この点は本サービスの差別化ポイントと言えるでしょう。
世界中のユーザーの投稿を閲覧できる点からSNSを想起するかもしれませんが、本サービスでは「いいね」や「コメント」などの評価機能をあえて設けていません。「良いこと」に優劣はありませんし、評価が新たな負担になることは、本サービスの方針と異なります。

さらに、過去の記録の振り返りを通して、日常の積み重なりを認識できるのも推しポイントですが、振り返り自体は一般的な日記サービスにも存在します。そこで、本サービスでは、振り返りには記録の蓄積が必要であることを念頭に置き、記録を積み重ねていけるよう、未投稿のリマインド通知機能や、他者の記録に触れる機会(世界中のユーザーの投稿一覧、Xでの共有)の提供、投稿内容を文章のみに限定するといった工夫をしています。

このように、本サービスは、一般的な日記アプリやSNSとは異なり、肯定的な出来事への気づきと、その積み重なりを実感する体験に特化した、新しい記録アプリです。

## 機能候補
### MVPリリース
・ユーザー登録
・ログイン
・投稿(本文のみ)
・自分の投稿一覧
・全投稿一覧(他者も含む)
### 本リリース
・未投稿リマインド通知
※現時点では、Rakeタスクを使用したメール通知による実装を想定している。  
└毎日あらかじめ設定した時間に通知を送ることで、記録することを思い出すきっかけを提供する
・Xでの共有
└他のユーザーの記録を見ることで、自分の一日を振り返るきっかけを得られ、記録を続ける動機づけにつながる
・キーワード検索機能
・記録一覧ページのページネーション
### 将来的なアップデート
・過去の記録のランダム表示
└1か月前の「良いこと」を表示し、日常の積み重なりを実感できる仕組み
・記録内容の傾向の可視化
└記録作成ページに、過去の投稿内容から抽出した「最近よく使われている言葉」をヒントとして表示することで、思いつかないという状況を防ぐ
